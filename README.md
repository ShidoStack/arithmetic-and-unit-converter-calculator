# Unit Converter & Calculator Web App

A fully responsive and modern Unit Converter + Calculator built using **HTML, CSS, and JavaScript**.

This project allows users to:
- Perform basic calculations
- Convert between different units
- Switch themes
- Use it smoothly on mobile and desktop

Perfect for beginners who want to understand DOM manipulation, event handling, and responsive UI design.

---

## Features

Basic Calculator  
- Addition  
- Subtraction  
- Multiplication  
- Division  

Unit Conversion  
- Length  
- Weight  
- Temperature  
- And more (depending on your implementation)

Modern UI
- Rounded buttons  
- Proper spacing  
- Mobile responsive design  
- Bottom navigation tabs  
- Theme toggle (Dark / Light mode)

## How It Works

1. HTML builds the structure.
2. CSS handles styling and responsiveness.
3. JavaScript:
   - Handles button clicks
   - Performs calculations
   - Converts units
   - Switches themes dynamically
